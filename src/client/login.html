We appreciate community contributions to code repositories open sourced by Renault. By signing a contributor license agreement, we ensure that the community is free to use your contributions.

Review the CLA document
The Renault Contributor License Agreement (CLA) document is available for review as a PDF.

Sign the CLA
When you contribute to a Renault open source project on GitHub with a new pull request, a bot will evaluate whether you have signed the CLA. If required, the bot will comment on the pull request, including a link to this system to accept the agreement.
